# IEx

Elixir's interactive shell

インタラクティブ環境の entory point

https://elixir-lang.readthedocs.io/en/new-guides/ref/master/iex/IEx/


# User switch command

`Ctrl + G` からコマンドを入力する。
新しいシェルを開始したり、リモートシェルへ接続できる

# Shell name

```sh
iex --sname foo
iex> node() # node名を表示(sname@HOST)
iex> Node.alive?()
```
